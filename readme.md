# AI Text Automation Demo

This project demonstrates a simple AI-style automation tool built with Python.
It converts raw text into a short summary and clear bullet points.

## What it does
- Reads text from user input or a text file
- Analyzes the text to identify important information
- Generates a short summary and key bullet points
- Saves the output to a file

## Why this is useful
This tool helps save time by automatically organizing and summarizing large amounts of text.

## How to run
1. Make sure Python is installed
2. Run the script:
   python main.py
3. Follow the on-screen instructions
